# Smart Event Popularity Predictor

## 📌 Overview

This project predicts the popularity of events using Machine Learning and Natural Language Processing (NLP).

## 🚀 Features

* Predicts event popularity (High, Medium, Low)
* Uses TF-IDF + Naive Bayes (text analysis)
* Uses KNN (structured data)
* Data visualization with charts & WordCloud

## 🛠️ Technologies Used

* Python
* Scikit-learn
* Pandas
* Matplotlib
* NLP (TF-IDF)

## 📊 Models Used

* Naive Bayes
* K-Nearest Neighbors (KNN)
